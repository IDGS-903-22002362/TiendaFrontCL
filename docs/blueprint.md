# **App Name**: León Fanshop

## Core Features:

- Home & Navigation Hub: Displays promotional banners, featured products, and category highlights, with a compact header for easy navigation to search, cart, and main menu, optimized for mobile-first experience.
- Product Listing & Filtering: Presents products in a responsive grid, enabling users to browse and filter by category, size, price, and availability, along with sorting options for an efficient shopping experience.
- Product Detail Page with AI Q&A Tool: Provides a comprehensive view of individual products, including a dynamic image gallery, detailed descriptions, pricing, variations, stock information, and an 'Add to Cart' button. Integrates an AI tool allowing users to ask natural language questions about the product and receive intelligent, context-aware answers.
- Shopping Cart Management: Allows users to review their selected items, adjust quantities, remove products, and view an updated subtotal, facilitating a seamless path towards checkout.
- Checkout Flow Prototype: A streamlined, multi-step user interface for entering shipping details and mock payment information, culminating in a clear order summary to demonstrate the purchase journey.
- API Data Integration Readiness: Establishes a modular frontend architecture to consume product, category, and order data via REST APIs from the existing Node.js + Express backend, designed for seamless integration with Firestore as the underlying database.

## Style Guidelines:

- Primary Color: Club León green (#007A53), reflecting official branding for headers, main buttons, and active elements. Chosen for its bold, iconic presence on a light background.
- Accent Color: Vibrant yellow (#FFC845), as specified by Club León's branding, used for secondary CTAs, badges, prices, promotions, and hover states to create clear highlights and visual interest.
- Text and Strong Contrast Elements: Charcoal black (#2D2926), providing strong legibility for text and reinforcing a premium feel against lighter backgrounds.
- Background Color: A very light, desaturated green (#C9EFE3), subtly echoing the primary green while maintaining a clean, unobtrusive canvas, as requested for a general clean background.
- Headlines and Titles: 'Space Grotesk' (sans-serif) for its strong, modern, and slightly techy aesthetic, aligning with the premium sports brand. Body Text: 'Inter' (sans-serif) for excellent legibility and neutrality, ensuring readability across all mobile devices.
- Utilize simple, modern vector icons with a thin-line aesthetic, ensuring high legibility and consistent visual language across all interface elements to complement the premium, clean design.
- Mobile-first responsive design featuring compact headers, large touch-friendly buttons and inputs, consistent spacing, and clear hierarchy. Product grids will adapt with two columns on mobile and more on larger screens, with sticky CTAs on product detail pages for improved usability.
- Implement subtle and performance-optimized micro-interactions for states such as loading, hover, and press, providing visual feedback without compromising fluidity or device performance.
